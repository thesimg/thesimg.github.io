Eleformer

A platformer where you collect elements and create substances
Should any loading problems arise, refresh the page.

KNOWN BUG: after combining one element, th eother elements do not collide.

Use the arrow keys to jump around

Collect elements and scroll through them using A and D

Drop the elements using SPACE

Combine the elements to make as many substances as you can

Press R to make a new level

This is based on the sticky platformer feature demo.

I made this game for a science project about the periodic table, it was made in five days.


Elements you can Collect:
Oxygen
Hydrogen
Carbon
Sodium
Potassium
Chlorine
Calcium
Aluminium

Substances you can make:
Table Salt - Na Cl
Water - H2 O
Baking Soda - Na H C O3
Bleach - Na Cl O
Cream of Tartar - K C4 H5 O6
Flour - C6 H10 O5
Eggshell - Ca C O3
Drain Cleaner - Na O H
Aluminum Foil - Al2 O3
Carbon Dioxide - C O2
